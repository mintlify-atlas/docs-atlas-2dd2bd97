---
title: "Update inventory"
description: "Directly update the quantity and stock thresholds for an inventory record."
---

<Note>
  All endpoints require an `Authorization: Bearer <token>` header.
</Note>

## Endpoint

```
PUT /api/inventory/:id
```

<Warning>
  It is recommended to use stock movements (`POST /api/stock-movements`) to update quantities. That approach records every change in the movement log and provides a full audit trail. Direct inventory updates via this endpoint bypass the movement log entirely.
</Warning>

## Path parameters

<ParamField path="id" type="string" required>
  UUID of the inventory record to update. This is the inventory record ID, **not** the product ID.
</ParamField>

## Request body

All fields are optional. Only provided fields will be updated.

<ParamField body="quantity" type="number">
  New stock quantity on hand.
</ParamField>

<ParamField body="minStock" type="number">
  Minimum stock threshold for low-stock alerts.
</ParamField>

<ParamField body="maxStock" type="number">
  Maximum stock threshold.
</ParamField>

## Response

Returns the updated inventory record.

<ResponseField name="id" type="string" required>
  Unique identifier for the inventory record (UUID).
</ResponseField>

<ResponseField name="productId" type="string" required>
  UUID of the product this inventory record tracks.
</ResponseField>

<ResponseField name="quantity" type="number" required>
  Updated stock quantity on hand.
</ResponseField>

<ResponseField name="minStock" type="number" required>
  Minimum stock threshold.
</ResponseField>

<ResponseField name="maxStock" type="number" required>
  Maximum stock threshold.
</ResponseField>

<ResponseField name="tenantId" type="string" required>
  UUID of the tenant this inventory record belongs to.
</ResponseField>

<ResponseField name="createdAt" type="string" required>
  ISO 8601 timestamp of when the record was created.
</ResponseField>

<ResponseField name="updatedAt" type="string" required>
  ISO 8601 timestamp of when the record was last updated.
</ResponseField>

## Examples

<CodeGroup>

```bash curl
curl --request PUT \
  --url http://localhost:5000/api/inventory/c3d4e5f6-a7b8-9012-cdef-345678901234 \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{
    "quantity": 200,
    "minStock": 25,
    "maxStock": 600
  }'
```

```javascript fetch
const response = await fetch(
  'http://localhost:5000/api/inventory/c3d4e5f6-a7b8-9012-cdef-345678901234',
  {
    method: 'PUT',
    headers: {
      'Authorization': 'Bearer <token>',
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({
      quantity: 200,
      minStock: 25,
      maxStock: 600,
    }),
  }
);

const inventory = await response.json();
```

</CodeGroup>
